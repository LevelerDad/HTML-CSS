Revisando CSS

Tipos de Estilos em CSS

  • Inline

    • Só na linha.

  • Interno

    • Só na página.

  • Externo

    • Para tudo.

    Fazer um style.css, coloco a tag do link:css, crio o link no head, após o title e clico no nome criado leo link.css, aí o VSCode vai criar um arquivo.css no qual vou colocar os seletores.

    Primeira coisa que coloco no CSS Externo é a regra da acentuação @charset "UTF-8";. Não esquecer o ; após cada regra.

Psicologia das cores

Representação por Código RGB. Coloco rgb(0,0,255) para azul. RGB vai de 0 a 255. Com vírgula e entre parenteses.

Representação por Código Hexadecimal. Coloco #0000ff. Hexadecimal tem 16 algarismos. Vai de 0 a 9 e de a a f. Coloco tudo sem vírgula e sem parenteses.

Fazendo Gradiente

CSS entende gradiente como imagem. Então coloco background-image: linear-gradient(45deg, white, red);. Não esquecer de separar com espaço e vírgula. Colocar background-attachment: fixed para a página não ficar rolando.

Posso colocar ...radial-gradient(circle, white, red);. Posso ainda colocar quanto de cor quero para cada uma (..., white 10%, red 50%);.
